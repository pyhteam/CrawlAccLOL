====================================================
HƯỚNG DẪN SỬ DỤNG ỨNG DỤNG CRAWL TÀI KHOẢN LOL
====================================================

1. CHẠY ỨNG DỤNG:
   - Mở Command Prompt hoặc Terminal
   - Di chuyển đến thư mục chứa file: cd C:\Users\Sen\Desktop\CrawlAccLOL
   - Chạy lệnh: python app_crawl_lol.py

2. SỬ DỤNG MENU:
   
   Option 1 - Crawl tài khoản mới:
   - Nhấn phím 1 và Enter
   - Ứng dụng sẽ tự động crawl 6 trang từ chothuesub.com
   - Mất khoảng 10-15 giây để hoàn thành
   - Dữ liệu sẽ được lưu vào file chothuesub_accounts.json
   - File CSV cũng được tạo để xem trong Excel
   
   Option 2 - Xem danh sách đã crawl:
   - Nhấn phím 2 và Enter
   - Hiển thị 20 tài khoản đầu tiên
   - Hiển thị thống kê theo rank và giá
   
   Option 3 - Sắp xếp theo giá (thấp -> cao):
   - Nhấn phím 3 và Enter
   - Hiển thị 30 tài khoản có giá thấp nhất
   - Giúp tìm các tài khoản giá rẻ
   
   Option 4 - Thoát:
   - Nhấn phím 4 và Enter
   - Thoát khỏi ứng dụng

3. LƯU Ý:
   - Lần đầu chạy cần chọn Option 1 để crawl dữ liệu
   - Dữ liệu được lưu tự động, lần sau mở lại sẽ load dữ liệu cũ
   - Nếu muốn cập nhật dữ liệu mới, chọn Option 1 lại
   - File CSV có thể mở bằng Excel để xem chi tiết

4. CÁC FILE ĐƯỢC TẠO:
   - chothuesub_accounts.json: File dữ liệu chính
   - chothuesub_accounts.csv: File để xem trong Excel
